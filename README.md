# 📮 우편봉투 인쇄 시스템

우편봉투를 자동으로 생성하는 웹 애플리케이션입니다.

## 🚀 실행 방법

### 1. 패키지 설치

```bash
pip install -r requirements.txt
```

### 2. 웹앱 실행

```bash
streamlit run app.py
```

브라우저가 자동으로 열리며 `http://localhost:8501`에서 웹앱을 사용할 수 있습니다.

## 📋 사용 방법

1. **엑셀 파일 업로드**: 상호와 금액 정보가 포함된 엑셀 파일(5.xlsx 형식)을 업로드합니다.

2. **추가 텍스트 설정**:
   - 봉투에 표시할 추가 텍스트를 입력합니다 (예: "감사합니다")
   - 글씨 크기를 조절합니다 (8~30)
   - 글씨 색상을 선택합니다

3. **처리**: "데이터 정렬 및 PDF 생성" 버튼을 클릭합니다.

4. **다운로드**: 
   - 정렬된 엑셀 파일 다운로드
   - 우편봉투 PDF 파일 다운로드

## 📁 필요한 파일

프로젝트 폴더에 다음 파일들이 있어야 합니다:

- `number.xlsm` - 상가별 상호 순서 정보
- `g.jpg` - 봉투에 표시될 로고 이미지
- `C:/Windows/Fonts/H2GTRE.TTF` - 한글 폰트

## 📊 입력 파일 형식

업로드하는 엑셀 파일은 다음 컬럼을 포함해야 합니다:

- `상호` - 상호명
- `금액` 또는 `입금금액` - 금액 정보

## 🎨 기능

- ✅ 엑셀 파일 자동 정렬 (number.xlsm 기준)
- ✅ 상가별 순서대로 데이터 정리
- ✅ 우편봉투 PDF 자동 생성
- ✅ 로고 및 브랜드명 자동 삽입
- ✅ 커스텀 텍스트 추가 가능
- ✅ 글씨 크기 및 색상 조절 가능
- ✅ 정렬된 엑셀 파일 다운로드
- ✅ 사용하기 쉬운 웹 인터페이스

## 🛠️ 기술 스택

- **Streamlit** - 웹 인터페이스
- **Pandas** - 데이터 처리
- **ReportLab** - PDF 생성
- **OpenPyXL** - 엑셀 파일 처리
